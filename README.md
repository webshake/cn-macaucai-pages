# cn-macaucai-pages

本仓库用于归档和发布多个独立的 HTML 页面。页面内容不针对任何特定域名或网站，仅供存档和分发用途。

## 项目简介

`cn-macaucai-pages` 是一个静态 HTML 页面集合仓库。其中包含的页面均为独立、自包含的 HTML 文件，可用于历史记录、资料备份或轻量级展示。仓库不绑定任何特定网络服务或商业项目。

## 目录结构

```
├── README.md          # 本文件
├── index.html         # 可选：仓库入口页面
├── pages/             # 归档的 HTML 页面目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 页面引用的静态资源（CSS、JS、图片等）
    ├── style.css
    ├── script.js
    └── images/
```

- `pages/`：存放所有独立 HTML 页面。每个页面应能独立运行，不依赖外部动态服务。
- `assets/`：存放页面共享的静态资源。资源文件应尽量保持轻量，避免引入第三方外部链接。

## 页面归档说明

- 所有页面均为静态 HTML，不含服务器端脚本。
- 页面可能包含简单的 JavaScript 和 CSS，但不应包含外部 API 调用或外部资源（除非明确注明）。
- 页面内容遵循通用归档规范，不包含任何营销、诱导点击或商业推广信息。
- 若需要引用外部资源（如字体、库），建议使用本地副本或注明来源。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面需通过 Pull Request 提交。
- 欢迎提交 issue 报告页面问题或提出改进建议。
- 页面内容应保持简洁、中立，不涉及敏感信息或违反 GitHub 社区准则的内容。
- 若页面引用了第三方内容，请确保遵守相应版权和许可要求。
